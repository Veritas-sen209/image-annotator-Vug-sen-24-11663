# Project Documentation

## Workflow
1. Load an image from your computer
2. Select annotation tool (rectangle, line, circle)
3. Draw on the image using the mouse
4. Measure distance between two points
5. Save annotated image

## Limitations
- Measurements are in pixels only
- Undo/redo not implemented
- No real-world calibration

## Future Improvements
- Add color picker
- Real-world calibration
- Undo/redo functionality
- Add more annotation tools
